Data Collection and Usage

Root Color Picker is designed with privacy in mind. We want to be clear about what data our extension handles:

Data We Store Locally

  • Recent color picks
  • Theme preference (light/dark mode)

How We Store Data

  • All data is stored locally in your browser using Chrome's storage API
  • No data is ever transmitted to external servers
  • No data is shared with any third parties

Purpose of Data Storage

We store this data solely to:

  • Show your recently picked colors for easy access
  • Remember your preferred theme setting

Data Control

You have full control over your data. You can:

  • Clear all stored data using the "Clear All" button in the extension
  • Remove all data by uninstalling the extension

Changes to Privacy Policy

We will update this privacy policy as needed. Please review periodically.
